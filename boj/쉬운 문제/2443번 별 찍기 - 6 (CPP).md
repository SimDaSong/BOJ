#### 문제

첫째 줄에는 별 2×N-1개, 둘째 줄에는 별 2×N-3개, ..., N번째 줄에는 별 1개를 찍는 문제

별은 가운데를 기준으로 대칭이어야 한다.

#### 입력

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

#### 출력

첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.



#### 예제 입력 1

```
5
```

#### 예제 출력 1

```
*********
 *******
  *****
   ***
    *
```



## 2. 내 소스 코드

```C++
#include <iostream>
int main() {
	int n;
	scanf("%d", &n);

	for (int i = 0; i < n; i++) {
		for (int j = 0; j < i; j++) printf(" ");
		for (int j = 0; j < 2 * (n - i) - 1; j++) printf("*");
		printf("\n");
	}
}
```
