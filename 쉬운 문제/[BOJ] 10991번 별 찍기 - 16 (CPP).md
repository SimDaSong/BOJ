#### 문제

#### 입력

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

#### 출력

첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.

## 2. 내 소스 코드

```C++
#include <iostream>

int main() {
	int n;
	scanf("%d", &n);

	for (int i = 0; i < n; i++) {
		for (int j = i + 1; j < n; j++) printf(" ");
		for (int j = 0; j <= i; j++) printf("* ");
		printf("\n");
	}
}
```
