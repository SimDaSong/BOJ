#### 문제

예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.

#### 입력

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

#### 출력

첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.

## 2. 내 소스 코드

```C++
#include <iostream>

int main() {
	int n;
	scanf("%d", &n);

	if (n == 1) {
		printf("*"); return 0;
	}

	// 첫째 줄
	for (int i = 0; i < n - 1; i++) printf(" ");
	printf("*\n");
	// 중간 줄
	for (int i = 1; i < n - 1; i++) {
		for (int j = i + 1; j < n; j++) printf(" ");
		printf("*");
		for (int j = 0; j < i * 2 - 1; j++) printf(" ");
		printf("*\n");
	}
	// 마지막줄
	for (int i = 0; i < n * 2 - 1; i++) printf("*");
}
```
