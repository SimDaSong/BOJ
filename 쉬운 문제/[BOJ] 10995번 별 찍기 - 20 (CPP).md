#### 문제

예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.

#### 입력

첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.

#### 출력

첫째 줄부터 차례대로 별을 출력한다.

```
1
```

```
*
```


```
* *
 * *
```

```
3
```

```
* * *
 * * *
* * *
```

#### 예제 입력 4

```
4
```

```
* * * *

* * * *

```



## 2. 내 소스 코드

```C++
#include <iostream>

int main() {
	int n;
	scanf("%d", &n);

	for (int i = 0; i < n; i++) {
		for (int j = 0; j < n; j++) {
			if (i % 2) printf(" *");
			else printf("* ");
		}
		printf("\n");
	}
}
```
