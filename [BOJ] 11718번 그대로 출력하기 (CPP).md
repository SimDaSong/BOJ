#### 문제

입력 받은 대로 출력하는 프로그램을 작성하시오.

#### 입력

입력이 주어진다. 입력은 최대 100줄로 이루어져 있고, 알파벳 소문자, 대문자, 공백, 숫자로만 이루어져 있다. 각 줄은 100글자를 넘지 않으며, 빈 줄은 주어지지 않는다. 또, 각 줄은 공백으로 시작하지 않고, 공백으로 끝나지 않는다.

#### 출력

입력받은 그대로 출력한다.



#### 예제 입력 1

```

Baekjoon
Online Judge
```

```C++
/*
	while(ch!=EOF) {
		ch = getchar();
		putchar(ch);
	}

	로 코드를 짜는 경우, EOF까지 출력하기 때문에 틀림
*/

#include <iostream>

int main() {
	char ch = 0;

	while (true) {
		ch = getchar();
		if (ch == EOF) return 0;
		putchar(ch);
	}

	return 0;
}
```
